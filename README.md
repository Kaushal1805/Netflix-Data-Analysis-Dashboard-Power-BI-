# 🎬 Netflix Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Netflix Data Analysis Dashboard** built using **Power BI**.
It provides insights into movies and TV shows based on ratings, genres, countries, release years, and directors.

The goal of this project is to demonstrate **data visualization, storytelling, and business insights extraction** using real-world entertainment data.

---

## 🚀 Key Features

* 📅 **Year of Release Filter** – Analyze content trends over time
* 🎭 **Content Type Filter** – Movie vs TV Show comparison
* 🌍 **Country Filter** – Explore content distribution across countries
* ⭐ **Ratings Analysis** – Most common content ratings (TV-MA, TV-14, etc.)
* 🎬 **Genre Distribution** – Popular genres on Netflix
* 🎥 **Top Directors** – Most frequent content creators
* 📈 **Release Trend Analysis** – Year-wise content growth
* 🌎 **Country-wise Titles** – Content contribution by countries

---

## 📊 Dashboard Insights

* Majority of content is **Movies compared to TV Shows**
* **TV-MA and TV-14** are the most common ratings
* Content production increased significantly after **2015**
* Certain countries dominate Netflix content production
* Genres like **Action, Drama, and Documentaries** are highly popular

## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization & dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculations and measures
* **Dataset** – Netflix titles dataset

## 📂 Dataset Details

The dataset includes:

* Country

* Rating

## ⚙️ How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Interact with filters (slicers) like:

   * Year of Release
   * Content Type
   * Country
4. Explore insights visually
